Installing sec-wall on Debian

{% markdown %}
  • Most of the dependencies can be apt-installed {% syntax text %} # apt-get install python-pip python-pesto python-gevent python-zdaemon python-yaml python-lxml {% endsyntax %}
  • For the time being, Spring Python needs to be installed via pip {% syntax text %} # pip install springpython {% endsyntax %}
  • Everything's ready for installing sec-wall now {% syntax text %} # pip install sec-wall {% endsyntax %}
  • That's it, head on to the documentation and have fun!
