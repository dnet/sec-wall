sec-wall

A feature packed high-performance security proxy

Welcome to sec-wall

sec-wall is a high performance security proxy featuring SSL/TLS, WSSE and HTTP Auth Basic/Digest with powerful URL matching/rewriting and an optional headers enrichment. It's a security wall you can conveniently fence the otherwise defenseless backend servers with.
